# **App Name**: Internship AI Match

## Core Features:

- Dataset Upload: Upload mock datasets of students and companies in JSON format.
- AI Matching Engine: Matches students to internship opportunities using TF-IDF, Cosine Similarity, Jaccard Similarity and includes a fairness adjustment tool.
- Fairness Boost: Applies a fairness boost to candidates from SC/ST/PwD backgrounds.
- Ranked Shortlists: Generates ranked shortlists of students for each company, displaying similarity scores and highlighting fairness-boosted candidates.
- Summary Metrics Display: Display summary metrics showing representation of rural students and SC/ST/PwD candidates.
- Dynamic Capacity: Take capacity limits of the companies into account, and recommend an appropriate number of internship matches to each.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to represent innovation and trust.
- Background color: Light gray (#F0F0F0) to ensure readability and a modern feel.
- Accent color: Green (#2ecc71) to highlight fairness-boosted candidates and positive metrics.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, neutral, readable style.
- Use clear, tabular layouts for displaying datasets and results, with intuitive navigation.
- Employ simple, professional icons to represent different categories (e.g., location, sector, fairness).
- Subtle transitions when displaying shortlists and updating metrics.
